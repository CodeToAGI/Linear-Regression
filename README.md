## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/CodeToAGI/ml-from-scratch.git
cd ml-from-scratch

# 2. Create virtual environment
python -m venv myenv
myenv\Scripts\activate     # Windows
# source myenv/bin/activate # Linux/Mac

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run any episode solution
cd episodes/ep78
python ep78_challenge_solution.py
